# Food Ordering Website

This is a simple food ordering website created using HTML and CSS. The project contains a homepage with order details, contact information, and 4 delicious dish images. It is designed with a clean and user-friendly interface.

## Features
- Simple and responsive homepage
- Food order section
- Contact Us details
- 4 food dish images
- Clean design using HTML and CSS

## Technologies Used
- HTML5
- CSS3

## How to View
1. Clone or download this repository
2. Open the project folder
3. Run the `index.html` file in your browser
